# -Eco-Sensing---Simulateur-de-Reseau-de-Capteurs-Contraints
L'objectif principal de ce projet est de  développer un logiciel de simulation pour un nouveau type de capteur agricole autonome (IoT). Ce capteur doit être capable de mesurer des données (température/humidité), de les stocker dans une mémoire tampon limitée  (buffer), et de les transmettre à une station de base.
